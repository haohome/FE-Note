## 1.类
 - 类是对象的类型
 - 类具有相同属性和方法的一组对象的集合
 
 1. 定义:确定对象将拥有的特征(属性)和方法(行为)
 2. 类的组成:属性和方法
 3. 定义类的步骤
    > a.定义类名
    > b.编写类的属性
    > c.编写类的方法
 ```Java
 public class 类名{
   //定义属性部分(成员变量)
   属性1的类型 属性1;
   属性2的类型 属性2;
   //定义方法部分
   方法1;
   方法2;
 }
 ```
## 2.对象
### 对象的属性
 - 对象具有的各种特征
 - 每个对象的每个属性都拥有特定值
### 对象的方法
 - 对象可以执行的操作

使用对象的步骤:
 a. 创建对象(new)
 b. 使用对象
    引用对象的属性:对象名.属性
    引用对象的方法:对象名.方法
> 类是抽象的概念,对象是一个具体的实物
